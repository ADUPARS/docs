Global Arguments

Redux has various levels of arguments, one for each type of object. So it makes sense that there would be global arguments which trickle down to all aspects of your project.

Arguments affect all aspects of Redux

You'll need to spend some time famililiarizing yourself with the different argument types for each object. For an extensive listing of all global arguments visit our Global Arguments page.

Select a Unique opt_name

Redux is built to allow multiple projects to use Redux at once. You must select a unique opt_name to avoid avoid collision with other projects. Make sure you choose something that's not too common. Oh, and no spaces.
